# AdMob library for Godot 
This is an Android AdMob plugin for Godot Game Engine 3.2.2 and higher.

[![Donate](https://img.shields.io/badge/Donate-PayPal-informational.svg)](https://paypal.me/cgisca)


### Supported features:
- Banner ads
- Interstitial ads
- Rewarded ads
- Consent info
